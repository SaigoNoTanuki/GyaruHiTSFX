A R.E.P.O. mod that plays a "Gyaru"(Ado) sound after being hit.

It's a bit janky, Will update in the future if I see a need for it.

Dependencies:
BepInEx 5.
REPOLib

How to use:
-Install zip file.
-Extract zip.
-Paste unzipped folder with the .dll and AssetBundle in BepInEx/plugins. (Usually in the game's root directory)
-Launch the game.
 OR,
-Use a Mod Loader like R2ModMan.
